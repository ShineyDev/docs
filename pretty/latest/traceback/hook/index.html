Hook

pretty.traceback.hook(cls=None, **kwargs)

Hooks pretty.traceback into the current Python session.

Note

You can set the PYTHONPRETTYTRACEBACK environment variable to a truthy value to hook pretty.traceback into all Python sessions.

Warning

This will replace attributes and functions in the traceback module with attributes and methods from the formatter.

The following cannot change when a TracebackFormatter subclass overrides built-in traceback methods:

  • Any parameter kind, name, order, or type of any function.

  • The return or yield type of any function.

The following can change when a TracebackFormatter subclass overrides built-in traceback methods:

  • The number and content of lines yielded by any format_* function.

  • The content printed by any print_* function.

Parameters

  • cls (Type[TracebackFormatter]) – The formatter class to use. Defaults to PrettyTracebackFormatter.

  • **kwargs – Keyword arguments are passed to TracebackFormatter.__init__.

Returns

The formatter.
