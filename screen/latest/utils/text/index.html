screen.utils.len(s)¶

Calculates the length of a string. This function takes into account CJK and zero-width characters.

Note

The string passed to this function is expected to have been normalized.

Parameters

s (str) – The string to calculate the length of.

Returns

The calculated length of the string.

Return type

int

Examples

>>> len("oranges")
7

>>> len("\x00")
0

>>> len("日本語")
6

>>> len("\u0061\u0301")  # á
2

>>> len(normalize("\u0061\u0301"))  # á
1
screen.utils.normalize(s)¶

Normalizes a string.

This function does the following, in order:

  • Calls unicodedata.normalize("NFC", ...).

  • Normalizes line endings.

  • Expands horizontal tabs with a width of four columns.

  • Expands vertical tabs with a height of one row, and emulates carriage return.

  • Emulates backspace and delete.

Parameters

s (str) – The string to normalize.

Returns

The normalized string.

Return type

str

Examples

>>> normalize("\u0061\u0301")  # á
"\u00E1"

>>> normalize("cool text\vtext")
"cool text\r\n         text"

>>> normalize("text text\rcool")
"cool text"
