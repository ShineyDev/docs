measure(h, w)

Calculates the desired size of the control. This method is a cached implementation of measure_core().

This method’s parameters, raises, and returns are identical to measure_core()

abstractmethod measure_core(h, w)

Calculates the desired size of the control.

Parameters

  • h (Optional[int]) – The available height. Can be None when the parent wants to measure the child. This is a soft constraint and this method may return a smaller or larger integer and hope the parent can accommodate.

  • w (Optional[int]) – The available width. Can be None when the parent wants to measure the child. This is a soft constraint and this method may return a smaller or larger integer and hope the parent can accommodate.

Returns

The desired size of the control.
