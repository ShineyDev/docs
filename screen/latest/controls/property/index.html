Control Property¶

screen.controls.property(type, default, required, invalidate_measure, invalidate_render, doc=None)¶

Registers a property on a Control.

Parameters

  • type (Type[Any]) – The value type.

  • default (Any) – The default value for the property. This can be accessed, overwritten, and overridden by inheriting classes as default_*.

  • required (bool) – A boolean indicating whether the property should be a required parameter to Control.__init__ and a non-nullable type match in the property setter.

  • invalidate_measure (Union[bool, Callable[[Any, Any], bool]]) – A boolean (or a callable taking the current and modified values and returning a boolean) indicating whether modifying the property should invalidate cached measures.

  • invalidate_render (Union[bool, Callable[[Any, Any], bool]]) – A boolean (or a callable taking the current and modified values and returning a boolean) indicating whether modifying the property should invalidate cached renders.

  • doc (str) – The property documentation. Defaults to None.

Examples

class Text(Control):
    content = property(str, None, True, True, True)

class Border(Control):
    header = property(Optional[str], None, False, lambda b, a: len(b) != len(a), True)
