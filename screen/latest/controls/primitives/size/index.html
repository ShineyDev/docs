Size¶

class screen.controls.primitives.Size(value)¶

Represents the height or width of a cell.

Parameters

value (Union[int, str]) – Can be “auto”, an integer, or a star-size; for example, “*” or “4*”. A value of “*” is equal to “1*”.

x == y
x != y

Compares two Size objects.

hash(x)

Returns the hash of the Size object.

property is_auto¶

Whether the size is automatically-sized. An automatically-sized cell inherits its size from its child control.

Type

bool

property is_star¶

Whether the size is star-sized. A star-sized cell is calculated as a weighted proportion of available space. The weight is value.

Type

bool

property value¶

The value of the size.

Type

int
