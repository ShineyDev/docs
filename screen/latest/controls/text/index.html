Text¶

class screen.controls.Text(**kwargs)¶

Represents a control used to display text.

Parameters

  • content (str) – The text’s content. This parameter is required.

  • background (Color) – The control’s background.

  • case (Case) – The text’s case.

  • foreground (Color) – The control’s foreground.

  • height (int) – The control’s height.

  • horizontal_alignment (HorizontalAlignment) – The control’s horizontal alignment.

  • horizontal_text_alignment (HorizontalAlignment) – The text’s horizontal text alignment.

  • layer (int) – The control’s layer.

  • margin (Thickness) – The control’s margin.

  • max_height (int) – The control’s max height.

  • max_width (int) – The control’s max width.

  • min_height (int) – The control’s min height.

  • min_width (int) – The control’s min width.

  • padding (Thickness) – The control’s padding.

  • style (Style) – The control’s style.

  • trim_boundary (Boundary) – The text’s trim boundary.

  • vertical_alignment (VerticalAlignment) – The control’s vertical alignment.

  • vertical_text_alignment (VerticalAlignment) – The text’s vertical text alignment.

  • width (int) – The control’s width.

  • wrap_boundary (Boundary) – The text’s wrap boundary.

measure(h, w)¶

Calculates the desired size of the control. This method is a cached implementation of measure_core().

This method’s parameters, raises, and returns are identical to measure_core()

measure_core(h, w)¶

Calculates the desired size of the control.

Parameters

  • h (Optional[int]) – The available height. Can be None when the parent wants to measure the child. This is a soft constraint and this method may return a smaller or larger integer and hope the parent can accommodate.

  • w (Optional[int]) – The available width. Can be None when the parent wants to measure the child. This is a soft constraint and this method may return a smaller or larger integer and hope the parent can accommodate.

Returns

The desired size of the control.

Return type

Tuple[int, int]

render(h, w)¶

Renders the control. This method is a cached implementation of render_core().

This method’s parameters, raises, and returns are identical to render_core().

render_core(h, w)¶

Renders the control.

Parameters

  • h (int) – The available height. This is a hard constraint and the iterator returned by this method is expected to yield a correctly sized block.

  • w (int) – The available width. This is a hard constraint and the iterator returned by this method is expected to yield a correctly sized block.

Returns

An iterator yielding lines.

Return type
