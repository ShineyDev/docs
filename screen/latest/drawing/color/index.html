classmethod from_random(*, seed=None)¶

Constructs a random Color.

Tip

This method can create unsatisfactory colors. It is recommended to use from_random_hsl() or from_random_hsv() with custom-bound saturation and lightness/brightness values instead.

Parameters

seed (Union[bytearray, bytes, float, int, str]) – The seed to initialize the random number generator with.
